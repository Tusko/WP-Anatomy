Edit your alt very quick without reloading

== Description ==

The plugin add a field to images in Media Library. You can edit ALT text without reloading, with AJAX.
Enter text and save it with blur input field. So simple!

Compatible with `Qtranslate-X`.

&copy; [Arsmoon - Independent team](https://arsmoon.com/ "Arsmoon")

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ars-alt-editor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Media Library, choose 'List view', enter text in `Alternative Text` field
